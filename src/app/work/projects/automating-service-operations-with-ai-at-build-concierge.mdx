## What I Built

I engineered core systems to automate the entire customer and operations funnel:
- Developed an **LLM-based agent using GPT-4** for 24/7 lead intake and qualification via chat, SMS, and WhatsApp
- Integrated **Twilio** for omnichannel communication and webhook-based response handling
- Built **Stripe API automations** for deposit handling, balance settlement, and digital invoicing
- Connected **BigChange APIs** to sync bookings, job status, and mobile work tracking
- Implemented **voice fallback using ElevenLabs** for agent-guided call summaries and escalations
